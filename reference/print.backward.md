# Printing a `backward` object

The default print method for a `backward` object.

## Usage

``` r
# S3 method for class 'backward'
print(x, ...)
```

## Arguments

- x:

  A model object of class `backward` as produced by
  [`model_backward()`](https://nuwani-palihawadana.github.io/smimodel/reference/model_backward.md)
  function.

- ...:

  Other arguments not currently used.

## Value

No return value; called for side effects. Prints a summary of the fitted
model(s) to console.
