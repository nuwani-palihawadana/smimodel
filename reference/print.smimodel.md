# Printing a `smimodel` object

The default print method for a `smimodel` object.

## Usage

``` r
# S3 method for class 'smimodel'
print(x, ...)
```

## Arguments

- x:

  An object of class `smimodel`.

- ...:

  Other arguments not currently used.

## Value

No return value; called for side effects. Prints a summary of the fitted
model(s) to console.
